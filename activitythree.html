Day 01
Alternate Uses:

Alternate Uses is a divergent thinking exercise that fosters creative thinking. It's a fun and versatile activity suitable for groups of any size. This exercise not only warms up your brain but also enhances team cohesion, making team members more comfortable with each other before diving into focused brainstorming sessions.

How it works:

  1. Set the timer for 3 minutes by clicking the start button below.
  2. Pick an ordinary object, like a Vessel.
  3. Jot down as many ideas as you can of alternate ways you could use that object. For example, a vessel could be used as a pen holder, to store snacks, or use the vessel as a coin bank.
  4. If you're working in a group, go around the room and have each team member share their ideas.

Hint:

Select a random item and hit the set timer button and come up with multiple ideas as you can and if possible try with different people to come with maximum ideas in this time frame.
